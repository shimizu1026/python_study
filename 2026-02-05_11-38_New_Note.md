プログラムを動かすための「道具」を準備しています
例えるなら：

🏠 家を建てるために、工具や材材料を揃えている段階
🍳 料理を作るために、包丁やフライパンを用意している段階


🔧 具体的に何をインストールしているか
requirements.txt には以下の4つのライブラリが書かれています：
1️⃣ Flask（フラスク）
Flask==3.1.0

役割: Webアプリケーションを作るためのフレームワーク
具体的には: ブラウザで http://127.0.0.1:5000 にアクセスしたときに、画面を表示したり、ファイルをアップロードする機能を提供

2️⃣ pandas（パンダス）
pandas==2.2.0

役割: Excelファイルを読み書きするための強力なライブラリ
具体的には: .xlsx ファイルを開いて、データを抽出したり、加工したりする

3️⃣ openpyxl（オープンパイエックスエル）
openpyxl==3.1.2

役割: Excelファイルの細かい操作（セルの色、フォント、罫線など）を扱う
具体的には: 出力ファイルのヘッダーに色を付けたり、セルを結合したりする

4️⃣ Werkzeug（ヴェルクツォイク）
Werkzeug==3.0.1

役割: Flaskの補助ツール
具体的には: ファイルのアップロードを安全に処理する


🎯 なぜこれが必要なのか？
Pythonだけでは、Excelファイルを扱ったり、Webサーバーを動かすことができません。
だから、他の人が作った便利なプログラム（ライブラリ）を借りてくる必要があるんです。

📥 pip install コマンドの意味
bashpip install -r requirements.txt

pip: Pythonのライブラリをインターネットからダウンロードしてインストールするツール
install: インストールしてね
-r requirements.txt: requirements.txt に書いてあるライブラリを全部インストールして

つまり、**「必要な道具を全部揃えてください」**というお願いをしています。
ビルド済みのバイナリを使ってインストールしましょう。

pip install -r requirements.txt
エラーになる

🔧 解決方法：直接最新版をインストール
requirements.txt は無視して、以下のコマンドを実行してください：
bashpip install Flask pandas openpyxl Werkzeug
これで最新の安定版が自動的にインストールされます。バージョンを指定しないことで、**ビルド済みのバイナリ（.whl ファイル）**がダウンロードされるので、コンパイラ不要です！

📝 実行手順
ターミナルで以下を順番に実行：
1️⃣ まず pip を最新版にアップデート（推奨）
bashpython.exe -m pip install --upgrade pip
2️⃣ ライブラリをインストール
bashpip install Flask pandas openpyxl Werkzeug
これで成功するはずです！

💡 なぜこの方法が良いのか？

❌ pandas==2.2.0 → ソースコード（.tar.gz）をダウンロード → コンパイラが必要
✅ pandas (バージョン指定なし) → ビルド済みバイナリ（.whl）をダウンロード → すぐ使える

Python 3.13は新しいバージョンなので、古いバージョンのpandasはコンパイルが必要になってしまいます。

## 🌐 ブラウザでアクセス

サーバーが起動したら：

1. **Webブラウザ（Chrome、Edge、Firefoxなど）を開く**

2. **アドレスバーに以下を入力：**
```
   http://127.0.0.1:5000
```

3. **Enterキーを押す**

すると、美しいアップロード画面が表示されます！ 🎨

---

## 📱 画面の使い方

1. **ファイルをアップロード**
   - 画面中央にExcelファイルをドラッグ＆ドロップ
   - または、エリアをクリックしてファイルを選択

2. **「変換を開始」ボタンをクリック**

3. **変換が完了したら「ファイルをダウンロード」ボタンが表示される**
