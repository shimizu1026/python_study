## 関数の基本

**関数**とは、何度も使える処理のまとまりです。同じことを何回も書かなくていいメリットがあります。

```python
def greet(name):
    print("こんにちは、" + name + "さん！")

greet("太郎")   # → こんにちは、太郎さん！
greet("花子")   # → こんにちは、花子さん！
```

- `def` → 関数を作る合図
- 外から受け取る値を**引数**という。引数は関数を使う側が入力する
- `def`で関数を作るだけでは何も実行されない。**呼び出して初めて動く**

---

## printとreturnの違い

- `print` → 画面に表示する
- `return` → 結果を呼び出した場所に返す（保存して使えるようにする）

```python
def f(x):
    return x * 2

result = f(2)   # returnで返した値をresultに保存
print(result)   # → 4
```

returnは「関数の計算結果を外に持ち出すための出口」。returnがあっても`z = f(2)`のように変数に保存しないと使えない。

## スコープ（変数が使える範囲）

- **グローバル変数** → 関数の外で作った変数。どこからでも使える
- **ローカル変数** → 関数の中で作った変数。その関数の中でしか使えない

```python
x = 10   # グローバル変数

def f():
    y = 20   # ローカル変数
    print(x)  # ✅ 外の変数は中から見える

print(x)  # ✅
print(y)  # ❌ NameError！関数の中の変数は外から見えない
```

変数は**使う前に定義**しておかないとNameErrorになる。

## 条件分岐（if / else）

```python
if z == 5:
    print("z is 5")
else:
    print("z is not 5")
```

条件が合っているかどうかで実行される処理が変わる。

## 組み込み関数

Pythonに最初から用意されている関数。`def`で作らなくても使える。

| 関数 | 説明 | 例 |
|------|------|-----|
| `print()` | 画面に表示 | `print("hello")` |
| `len()` | 文字数を返す | `len("hello")` → 5 |
| `int()` | 整数に変換 | `int("123")` → 123 |
| `float()` | 小数に変換 | `float("3.14")` → 3.14 |
| `str()` | 文字列に変換 | `str(456)` → "456" |
| `abs()` | 絶対値を返す | `abs(-10)` → 10 |
| `max()` | 最大値を返す | `max(1, 5, 3)` → 5 |
| `min()` | 最小値を返す | `min(1, 5, 3)` → 1 |
| `input()` | キーボードから入力を受け取る | `input("名前：")` |
| `type()` | データの種類を返す | `type(42)` → int |

### float（浮動小数点数）について

浮動小数点数とは小数のこと。小数点の位置が動くことから「浮動」と呼ばれる。`float()`に変換できるのは数字として読める文字だけで、「あいう」などはエラーになる。

### input()について

- 受け取った値は**必ず文字列**になる
- 数字として使いたいときは`int()`や`float()`で変換が必要

```python
age = input("Enter your age:")
int_age = int(age)   # 文字列を整数に変換
if int_age < 21:
    print("You are young!")
else:
    print("Wow, you are old!")
```

## 必須引数とオプション引数

```python
def add_it(x, y=10):   # xは必須引数、y=10はオプション引数
    return x + y

add_it(2)      # x=2, y=10（デフォルト）→ 12
add_it(2, 3)   # x=2, y=3（上書き）→ 5
```

- **必須引数** → 渡し忘れるとエラーになる引数
- **オプション引数** → 渡さなくてもデフォルト値で動く引数。`=`でデフォルト値を指定する

## 例外処理（try / except）

エラーが起きてもプログラムが止まらずに対処できる仕組み。コードを書く側がエラーを予測してexceptに対処を書いておく。

```python
try:
    n = int(input("数字を入力してください："))
    print(10 / n)
except ValueError:
    print("数字を入力してください！")   # 文字を入力された場合
except ZeroDivisionError:
    print("0では割れません！")          # 0を入力された場合
except:
    print("何かエラーが起きました！")   # エラーの種類がわからない場合
```

- `try` → まずここを実行してみる
- `except` → エラーが起きたらここを実行する

## ドキュメンテーション文字列（docstring）

関数に説明書きをつける仕組み。`"""`で囲んで書く。HTMLやCSSのコメントアウトと考え方は同じ。

```python
def add(x, y):
    """
    xとyを足し算して返す関数です。

    :param x: int. 足す数
    :param y: int. 足す数
    :return: int. 足し算の結果
    :raises ValueError: 数字以外が渡された場合

    Example:
        >>> add(2, 3)
        5
    """
    return x + y
```

主に書く内容：
- 関数の概要（何をする関数か）
- `param` → 引数の説明
- `return` → 返り値の説明
- `raises` → エラーの説明
- `Example` → 使用例

`help(add)`と入力するとdocstringの内容を確認できる。
